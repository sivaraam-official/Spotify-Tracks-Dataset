# Spotify-Tracks-Dataset# Spotify Tracks ETL Pipeline & Analytics Dashboard

An end-to-end ETL (Extract, Transform, Load) pipeline built using Python, Pandas, SQLite, and Streamlit for processing and analyzing Spotify Tracks data.

## Features

* Automated ETL workflow
* Data cleaning and preprocessing
* Feature engineering for ML readiness
* SQLite-based data storage
* Interactive Streamlit dashboard
* SQL-powered analytics
* Downloadable ML feature dataset
* KPI monitoring and visualizations

## Tech Stack

* Python
* Pandas
* SQLite
* Streamlit
* Plotly Express
* SQL
* Git/GitHub

## Workflow

1. Extract Spotify dataset from Kaggle.
2. Clean and transform raw music data.
3. Store processed data in SQLite and CSV formats.
4. Visualize insights using Streamlit.
5. Export filtered datasets for machine learning models.

## Dashboard Insights

* Total Unique Tracks
* Genre Distribution
* Popularity Analysis
* Danceability vs Energy Correlation
* Genre-wise Performance Metrics
* ML Dataset Export Interface
